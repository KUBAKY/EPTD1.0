# WellMotion 功能测试计划

## 测试概述
本测试计划旨在全面验证 WellMotion 私人健身教练助手系统的所有功能模块，确保系统能够满足私教日常使用需求。

## 测试环境
- **前端**: http://localhost:3000
- **后端**: http://localhost:3001
- **数据库**: SQLite (本地文件)
- **测试设备**: Android 平板设备模拟
