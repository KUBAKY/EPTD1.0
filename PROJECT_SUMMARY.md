# WellMotion 项目开发总结

## 项目概述

## 部署说明

### 开发环境
1. 运行 `start-dev.bat` 脚本
2. 访问 http://localhost:3000
3. 使用默认账户登录

### 生产环境
1. 运行 `deploy/start-production.bat` 脚本
2. 访问 http://localhost:4173
3. 使用管理员账户登录

### Docker部署
1. 进入 `deploy` 目录
2. 运行 `docker-compose up -d`
3. 访问 http://localhost:4173
